Lesson One: The web and HTML

The internet and how it works

The internet or The World wide web is made up of a collection of HTML documents. Hypertext Markup Language or HTML for short is a language that browsers such as Google Chrome, Firefox, and Internet explorer use to inerpret web pages. Everytime you go to your favorite sites or any part of the internet, you are actually making a request via the internet to a server using the HTTP protocol. The server then resonds with the appropriate html document, using the HTTP protocol, that will be displayed on your browser.

These are the major pieces of the web

  • Client - A client is a computer and a browser where the web pages can be viewed
  • Internet - The vast network of computers
  • Servers - Computers that are used for hosing files
  • HTTP - Hypertext Transfer Protocol

Inline vs Block

There are two types of HTML elements Inline and block. A few inline elements that we learned this champer are b, em, img, and a tags. Block elements have a forcefield like box around them.
